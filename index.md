# A Hands-On Introduction to Using IIIF resources

## 27 May 2026 | Solving the IIIF-puzzle \- Images in Dialogue Helsinki Showcase

### Martin R. Kalfatovic, Managing Director, IIIF Consortium

**Description**  
This hands-on workshop will introduce the IIIF standard and how it can be used in the cultural heritage sector. Participants will be introduced to the core IIIF concept of the manifest, and learn how to find a IIIF manifest. Through working with IIIF accessible digital assets, the participants will learn to use key IIIF functions such as sharing a detail of an image via Universal Viewer, comparing objects in the Mirador viewer; create a virtual tour or quiz with Exhibit, and embed a viewer on a website. The primary focus will be on using images, but many of the concepts can also be applied to audio-visual and 3D content.

Prerequisites for the workshop include a basic knowledge of using web resources and cutting/pasting web URLs and content.
